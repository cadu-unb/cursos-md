## ABERTURA

*Imagine* / que você tem quinhentos hectares de lavoura. //
Sem sinal de internet. //
E um foco de incêndio começa / numa área que você *não consegue ver.* ///

*Quando você descobre...* // **já é tarde.** ↓

///

**Esse é o problema** / que o AgroLoRa resolve. ↓

---

## A SOLUÇÃO

O AgroLoRa é uma solução de monitoramento inteligente para propriedades rurais / baseada em tecnologia **LoRa** — //
uma rede de *longo alcance* / que funciona **sem depender de internet** / ou telefonia convencional. ↓

///

Instalamos sensores geolocalizados na propriedade / que monitoram continuamente / clima, / umidade / e fumaça. //
E enviam alertas em tempo real / *direto para o produtor.* ↓

///

O resultado prático: //
detecção **precoce** de incêndios, /
proteção de lavouras e pastagens, /
e decisões baseadas em dados — //
**mesmo nas regiões mais remotas do país.** ↓

---

## MERCADO

Nosso público são grandes propriedades com baixa conectividade, /
produtores expostos a riscos de incêndio /
e cooperativas que buscam uma gestão agrícola mais eficiente. ↓

//

Chegamos até eles por feiras agropecuárias, / rádio regional / e canais digitais. ↓

---

## MODELO DE NEGÓCIO

O modelo combina / venda do sistema, / instalação e configuração, / planos de manutenção — //
e projetos **personalizados** para grandes operações. ↓

//

Para entregar isso, / contamos com parceiros estratégicos: /
fabricantes de módulos LoRa, / fornecedores de sensores, /
cooperativas, / universidades / e órgãos de defesa civil rural. ↓

## IMPACTO

E além do impacto econômico para o produtor — ///

o AgroLoRa contribui diretamente /
para uma agricultura *mais produtiva e resiliente,* /
para a *prevenção de desastres climáticos,* /
e para a *proteção dos ecossistemas* que dependem do campo. ↓
