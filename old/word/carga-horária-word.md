## 2. RESUMO MATEMÁTICO DA CARGA HORÁRIA

### Fórmula de Cálculo

$$T_{total} = \sum_{i=1}^{n} t_i$$

Onde:
- $T_{total}$ = Tempo total do curso (minutos)
- $n$ = Número total de atividades
- $t_i$ = Tempo de cada atividade $i$

### Cálculo Detalhado

| Módulo | Tempo (min) | Tempo (horas) |
|--------|-------------|---------------|
| Módulo 0: Preparação | 44 | 0,73 |
| Módulo 1: Fundamentos | 47 | 0,78 |
| Módulo 2: Fluxo Markdown | 47 | 0,78 |
| Módulo 3: Edição de Estilo | 48 | 0,80 |
| Módulo 4: Estruturação Avançada | 80 | 1,33 |
| Módulo 5: Revisão Crítica | 51 | 0,85 |
| Módulo 6: Automação de Formatação | 61 | 1,02 |
| Módulo 7: Integração Avançada | 58 | 0,97 |
| Módulo 8: Segurança e Privacidade | 55 | 0,92 |
| Módulo 9: Controle de Qualidade | 46 | 0,77 |
| Módulo 10: Projeto Final | 61 | 1,02 |
| **TOTAL** | **598 min** | **9,97 horas** |

### Resultado

$$T_{total} = 44 + 47 + 47 + 48 + 80 + 51 + 61 + 58 + 55 + 46 + 61 = 598 \text{ minutos}$$

$$T_{total} = \frac{598}{60} = 9,97 \text{ horas} \approx 10 \text{ horas}$$

**Arredondado:** **10 horas** de carga horária total.
